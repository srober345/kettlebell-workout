# Kettlebell Workout Site

Multi-program training site with three science-based programs.

## Deployment
- Hosted on **GitHub Pages** (free, auto-deploys from `main` branch)
- Live URL: https://rwdavidson303.github.io/kettlebell-workout/
- GitHub Repo: https://github.com/rwdavidson303/kettlebell-workout

## Programs
1. **Kettlebell Program** (MWF) — 12 exercises, daily variation (Standard/Density/Power days)
2. **Abdominal Routine** (Tue/Thu/Sat) — 8 exercises, 3 rotating workouts
3. **Nutrition Plan** (Daily) — body recomposition macros, hotel meal plan, cafe options

## Key Files
- `index.html` — landing page, weekly schedule
- `monday/wednesday/friday.html` — daily workout pages
- `exercises.html` — exercise library with SVG illustrations
- `abs.html` — ab routine
- `nutrition.html` — nutrition plan
- `css/style.css` — full site styling
- `BUILD.md` — detailed project reference
